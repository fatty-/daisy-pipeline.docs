Objectives
-------------------------------------------------------------------------------

Create a simple Pipeline 2 module that  provides a script to query the Pipeline
2 Web Service  and get the current  list of jobs as a  nice human-readable HTML
report.


Contents of the directory
-------------------------------------------------------------------------------
 - readme.txt : this file
 + solution: the completed tutorial
   + job-lister: the module sources
   - org.daisy.pipeline.modules.job-lister.jar : the packaged Pipeline 2 module
   - makejar :  a utility script to build the module on Linux/Mac
 +  material: the base source files for the tutorial, with missing parts to
    complete


Getting started
-------------------------------------------------------------------------------

Follow the instructions in the companion slides:
