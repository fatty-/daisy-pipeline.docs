DAISY Pipeline 2 - Tutorial 1

Install and get started with the Pipeline 2 command line tool

February 16, 2011

Requisites

  1. Java jre - 1.6 http://www.java.com/en/download/manual.jsp
  2. Ruby - 1.8.7 http://rubyinstaller.org/downloads/
  3. Workshop daisy-pipeline build http://code.google.com/p/daisy-pipeline/

Keep an eye on the CLI documentation.

Local mode

Our goals:

  1. Get the CLI running.
  2. Inspect the help.
  3. Transform one document.
  4. Send a background job.
  5. Get the list of jobs.
  6. Get the status from a job.
  7. Delete a job.
  8. Be nice and shutdown the webservice.

Remote mode

We are going to achieve:

  1. Manually start the framework.
  2. Change the configuration file.
  3. Spot some differences between the remote and local CLI modes.
  4. Transform one document.
  5. Send a background job.
  6. Get job info and status.
  7. Retrieve the job result.
