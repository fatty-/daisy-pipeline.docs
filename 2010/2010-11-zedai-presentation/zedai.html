History

  • DAISY 2.02 (2001)
    • XHTML content
    • ncc.html
    • SMIL
  • DAISY 3
    • Shift toward XML
      • XHTML content
      • DAISY XML = DTBook
      • NCX, OPF
      • SMIL
    • standardized as the "ANSI/NISO Z39.86 Specifications for the Digital Talking Book"
    • revised (minor) as Z39.86-2005

DAISY 4

DAISY 4 = ZedNext

Major revision of Z39.86, in two parts:

  • ZedAI = Authoring and Interchange Framework
  • ZedDist = Distribution, most probably replaced by EPUB 3

Timeline

January 2011: Final Draft for Trial Use (aligned with EPUB)

April 2011: Final release

Key points

ZedAI = XML-based Framework

  • Adaptability
  • Modularity
  • Self-describing
  • Data repurposing

Minimal Document 

<document xmlns="http://www.daisy.org/ns/z3986/authoring/"
          xmlns:decl="http://www.daisy.org/z3986/2011/vocab/decl/#"
          xmlns:dcterms="http://purl.org/dc/terms/" xml:lang="en">
  <head>
    <meta rel="decl:profile" resource="http://www.daisy.org/z3986/2011/auth/profiles/genericdocument/1.0/" />
    <meta property="dcterms:identifier" content="daisy-z2011-exemplar-01" />
    <meta property="dcterms:publisher" content="DAISY Consortium" />
    <meta property="dcterms:date" content="2011-07-27T13:50:05-05:00" />        
  </head>
  <body>
    <h>Hello, world.</h>
  </body>
</document>

Profiles

  • Profile = XML grammar based on the ZedAI framework
  • made of schema modules
  • and optional features
  • all defined in a resource directory

Semantic Inflections

  • role attribute
    • values are CURIEs
    • vacabularies defined in RDF
  • inline RDF metadata

Example: MathML feature 

<document xmlns="http://www.daisy.org/ns/z3986/authoring/"
          xmlns:m="http://www.w3.org/1998/Math/MathML"
          xmlns:dcterms="http://purl.org/dc/terms/"
          xmlns:decl="http://www.daisy.org/z3986/2011/vocab/decl/#" xml:lang="en">
  <head>
    <meta rel="decl:profile" resource="http://www.daisy.org/z3986/2011/auth/profiles/book/1.0/">
      <meta rel="decl:feature" resource="http://www.daisy.org/z3986/2011/auth/features/mathml/1.0/" />
    </meta>
    <meta property="dcterms:identifier" content="daisy-z2011-pythagorean-001" />
    <meta property="dcterms:publisher" content="DAISY Consortium" />
    <meta property="dcterms:date" content="2005-07-27T13:50:05-05:00" />        
    
  </head>
  <body>
    <frontmatter>
      <section>
        <h>The Pythagorean Theorum</h>
      </section>
    </frontmatter>
    <bodymatter>
      <section>
        <h>The Pythagorean Theorum</h>
        <p>Pythagoras stated: 
          <m:math>
            <m:mi>c</m:mi>
            <m:mo>=</m:mo>
            <m:msqrt>
              <m:msup>
                <m:mi>a</m:mi>
                <m:mn>2</m:mn>
              </m:msup>
              <m:mo>+</m:mo>
              <m:msup>
                <m:mi>b</m:mi>
                <m:mn>2</m:mn>
              </m:msup>
            </m:msqrt>
          </m:math>
        </p>
      </section>
    </bodymatter>    
  </body>
</document>
