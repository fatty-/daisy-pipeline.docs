Tobi, in a nutshell

  • Authoring tool for talking books

    ("read aloud" e-books)

    • Human narration, not speech synthesis

    • Text synchronized with spoken words

  • Interactive audio workflow

    (unlike automated TTS production)

    • Live voice recording

    • Use pre-recorded tracks

    • Waveform editor

  • No XML editing

    (limited to minor text alterations)

 

 

 

 

Features highlight

  • Built-in content validator/checker

    (invalid markup, metadata errors, missing audio)

  • Helpers for publication metadata

    (Z39.86-2005 standard, soon EPUB3)

  • Content navigation

    (chapters, pages, document markup, authoring placemarks)

  • Unlimited undo/redo

    (+ "cleanup" function to save space)

 

Accessibility

  • Screen readers

    (tested with NVDA, JAWS 9+)

  • Built-in magnification

    (crisp vector graphics)

  • High-contrast mode

    (system settings)

  • Keyboard optimized

    (configurable shortcuts, quick focus change)

 

Open standards

  • DAISY 3 Digital Talking Books

    (synchronized DTBook + audio)
    ↪ fully-supported

  • EPUB 3 Media Overlays

    (synchronized (X)HTML5 + audio)
    ↪ may / June 2012

  • DIAGRAM image descriptions / annotations

    (standard not yet finalized)
    ↪ audio descriptions: April 2012

XML formats ⇄ great interoperability

 

Free and open source

  • Windows (XP, Vista, 7)

    (.NET 3.5 / 4.0, DirectX 9)

  • Install once, automatic new versions

    (9MB download, smaller updates)

  • Shared code, modular design

    (Urakawa SDK: Obi, Tobi)

  • Flexible licence

    (LGPL, BSD)
