Background

  • An increasing demand for accessible content

    • to a wide range of user groups

    • through various distribution channels

    • to an increasing variety of devices

    • in several output formats

  • Follow-up of the successful Pipeline 1 project

    • Pipeline 1: started in 2002, now in maintenance mode

    • Pipeline 2: started in 2010, total redesign

  • High-level objectives

    • Efficiency

      ↪ produce many documents in a short time

    • Low cost

      ↪ easy to develop and easy to maintain

    • Versatility

      ↪ several output formats

      ↪ interoperable with other systems

Approach

  • Modular system

    • easy to extend

    • easy to customize

    • open to commercial plugins/services

  • Promote "single source" publishing

Architecture

  • XProc core

    • "XML Pipeline Language"

    • W3C Recommendation since May 2010

    • several open source and commercial engines

  • Module system

    • XSLT tranforms | XPath functions | Java code | external tools

    • orchestrated via XProc + XML catalogs

  • Runtime framework

    • Java-based, powered by OSGi

    • Service-oriented

      ↪ Job management, logging, etc

Deployment options

  • Command line tool

  • Web service

    ➙ already available — gradually enriched

  • Web application

    ➙ June 2012

  • Desktop UI

    ➙ First half of 2013

Tobi, in a nutshell

  • Authoring tool for talking books

    • Human narration, not speech synthesis

    • Text, synchronized with spoken words

    ➙ "read aloud" e-books, illustrated children's stories

  • Interactive audio production

    • Live voice recording

    • Pre-recorded tracks

    • Waveform editor

    ➙ more time-consuming than automated TTS workflow

  • Limitation: no XML editing

    support for minor text corrections only

Open standards

  • DAISY 3 Digital Talking Books

    synchronized DTBook + audio

    ➙ fully-supported

  • EPUB 3 Media Overlays

    synchronized (X)HTML5 + audio

    ➙ may / June 2012

  • DIAGRAM image descriptions

    standard not finalized yet

    ➙ audio annotations: April 2012

XML formats ⇄ interoperability

Free and open source

  • Windows (XP, Vista, 7)

    .NET 3.5 / 4.0, DirectX 9

  • Install once, automatic new versions

    9MB download, smaller incremental updates

  • Shared code, modular design

    Urakawa SDK: Obi, Tobi

  • Flexible software licence

    LGPL, BSD

Features highlight

  • Built-in content validation

    invalid markup, metadata errors, missing audio

  • Publication metadata helpers

    Z39.86-2005, soon EPUB3

  • Rich navigation

    chapters, pages, document markup, placemarks, search

  • Unlimited undo/redo

    + "cleanup" function to save space

  • Flexible user-interface

    narrator's preferred text font/size, compact reading view, etc.

Accessibility

  • Screen readers

    tested with NVDA, JAWS 9+

  • Built-in magnification

    crisp vector graphics

  • High-contrast mode

    system settings

  • Keyboard-optimized

    configurable shortcuts, quick focus change
